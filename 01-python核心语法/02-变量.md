# 变量

## 1. 变量的定义

变量就是用来存储数据的。在编程语言中，将数据放入变量的过程叫做赋值。Python使用等号=作为赋值运算符，具体格式为：

```python
变量名 = 值(要存储的数据)
```



## 2. 变量命名规则与规范

**规则：**必须遵守，不遵守报错

- Python中变量名由字符、下划线和数字组成，且第一个字符不能是数字。

- Python中变量名不能包含空格、@、% 以及 $ 等特殊字符。

- Python中变量名的字母是严格区分大小写的，Age和age是不同的变量。

- Python中变量名不能和Python中的关键字相同，关键字是Python语言中一些已经被赋予特定意义的单词，这就要求开发者在开发程序时，不能用这些关键字作为变量名。

  | and   | as   | assert | break    | class  | continue |
  | ----- | ---- | ------ | -------- | ------ | -------- |
  | def   | del  | elif   | else     | except | finally  |
  | for   | from | False  | global   | if     | import   |
  | in    | is   | lambda | nonlocal | not    | None     |
  | or    | pass | raise  | return   | try    | True     |
  | while | with | yield  |          |        |          |

**规范：**建议，不遵守不会报错，但不符合业内通识

- 起名要有意义
- 如果变量名由多个单词组成建议使用：驼峰体、纯小写加下划线
