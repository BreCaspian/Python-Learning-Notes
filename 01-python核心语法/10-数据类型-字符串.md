# 数据类型-字符串

字符串，用来存储描述性质的数据。例如：姓名、地址、自我介绍等。字符串是一个有序的字符的集合

## 1. 类型转换

str()可以将任意数据类型转换成字符串类型



## 2. 字符串格式化

```python
# 方式一
print('我的名字是 %s, 我的年龄是 %s岁' % ('zzt', '4'))
# 方式二
print('我的名字是 {}, 我的年龄是 {}岁'.format('zzt', '4'))
# 方式三
name = 'zzt'
age = '4'
print(f'我的名字是 {name}, 我的年龄是 {age}岁'
```



## 3. 通用操作

- 相加：字符串 + 字符串
- 相乘：字符串 * 整数
- 获取长度：len()
- 索引取值
- 切片
- for循环取值
- 成员运算符：`数据 in 字符串` 如果在指定的字符串中找到值，返回bool类型。



## 4. 字符串的内置功能 

我们在开发程序时需要频繁对数据进行操作，为了提升我们的开发效率， python针对这些常用的操作，为每一种数据类型内置了一系列方法。我们现在要学习的是python为我们提供的这些方法

- 判断字符串是否以 XX 开头`startswith`/XX结尾`endswith`，得到一个布尔值

  ```python
  str2 = 'hello world'
  print(str2.endswith('L'))
  
  str3 = 'hello world'
  print(str3.endswith('L'))
  ```

- 判断字符串是否是数字组成`isdigit`，返回结果为True或False

  ```python
  str4 = '5201314'
  print(str4.isdigit())
  ```

- 去除字符串两边的 空格、换行符、制表符，得到一个新字符串

  ```python
  str1 = ' hello world' 
  print(str1.strip())
  ```

- 字符串变大写`upper`/小写`lower`，得到一个新字符串

  ```python
  str5 = 'My nAme is tonY！'
  print(str5.lower())  # 将英文字符串全部变小写
  print(str5.upper())  # 将英文字符串全部变大写
  ```

- 用新的字符替换字符串中旧的字符`replace`

  ```python
  str6 = 'my name is xxx, my age is 18!'  # 将xxx的年龄由18岁改成73岁
  print(str6.replace('18', '73'))  # 语法:replace('旧内容', '新内容')
  ```

- 字符串切割，得到一个列表

  ```python
  # 括号内不指定字符，默认以空格作为切分符号
  str8 = 'hello world'
  print(str8.split())
  
  # 括号内指定分隔字符，则按照括号内指定的字符切割字符串
  str9 = '127.0.0.1'
  print(str9.split('.'))
  ```

- 字符串拼接，得到一个新的字符串

  ```python
  data_list = ["1","2","3"]
  v1 = "_".join(data_list)
  print(v1)
  ```
