# 流程控制之分支结构

分支结构就是根据条件判断的真假去执行不同分支对应的代码。可分为单分支、双分支、多分支

## 1. 单分支

```python
if 条件:
    满足条件后要执行的代码块
```

- 条件可以是一个单一的值或者变量，也可以是由运算符组成的复杂语句。不管条件的结果是什么类型。if 都能判断它是否成立（真或者假）
- 代码块由具由相同缩进量的若干条语句组成。
- if语句的最后要有有冒号`:`，不要忘记。



## 2. 双分支

```python
if 条件:
    满足条件执行代码
else:
    if条件不满足就走这段
```



## 3. 多分支

```python
if 条件:
    满足条件执行代码
elif 条件:
    上面的条件不满足就走这个
elif 条件:
    上面的条件不满足就走这个
elif 条件:
    上面的条件不满足就走这个    
else:
    上面所有的条件不满足就走这段
```

- elif可以有0个或多个。
- else可以有0个或1个，且只能放在if语句的最后。



## 4. 条件表达式

语法：变量 = 结果1 if 条件 else 结果2

作用：根据条件(True/False) 来决定返回结果1还是结果2。
