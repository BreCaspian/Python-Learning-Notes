# 输入与输出

输出和输入也可理解为人和计算机的交互，人向计算机输入信息，计算机处理后再展示结果给用户。

## 1. 输入

在Python中使用input函数，从控制台获取用户输入的内容。

```python
变量 = input(tipmsg)  # 作用：将用户输入的内容赋值给变量
```

tipmsg 表示提示信息，它会显示在控制台上，告诉用户应该输入什么样的内容,如果不写 tipmsg，就不会有任何提示信息。

input会等待用户的输入，按下回车键后 input() 读取就结束了。用户输入任何内容，都存成字符串类型，然后赋值给等号左边的变量名



## 2. 输出

在Python中使用print函数，将括号中的内容显示在控制台中。

```python
print('hello world')  # 只输出一个值
print('first','second','third')  # 一次性输出多个值，值用逗号隔开
```

一次性输出多个值默认以空格隔开多个值，如果希望改变默认的分隔符，可通过sep参数进行设置。

```python
print('first', 'second', 'third', sep='|')
```

默认情况下，print输出之后是会换行的，这是因为print的end参数的默认值是“\n”，这个“\n”就代表了换行。如果希望print输出之后不会换行，则重设 end参数即可

```python
print(40,'\t',end＝"")
print(5O,'\t',end＝"")
print(60,'\t',end＝"")
```
