# 作用域

作用域，就是变量的有效范围，就是变量可以在哪个范围以内使用。有些变量可以在整段代码的任意位置使用，有些变量只能在函数内部使用。变量的作用域由变量的定义位置决定，在不同位置定义的变量，它的作用域是不一样的。

- Local局部作用域：函数内部。
- Enclosing 外部嵌套作用域 ：函数嵌套。 
- Global全局作用域：模块(.py文件)内部
- Builtin内置模块作用域：builtins.py文件。

在访问变量时，先查找本地变量，然后是包裹此函数外部的函数内部的变量，之后是全局变量，最后是内置变量。



## 1. global语句

默认情况下，在局部作用域对全局变量只能进行：读取和修改内部元素（可变类型），无法对全局变量进行重新赋值。如果想要在局部作用域中对全局变量重新赋值，则可以基于 `global`关键字实现。



## 2. nonlocal语句

在内层函数修改外层嵌套函数内的变量， 在被嵌套的内函数中进行使用。
