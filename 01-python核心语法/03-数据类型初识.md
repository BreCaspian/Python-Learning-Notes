# 数据类型初识

## 1. 整数类型

整数就是没有小数部分的数字，Python中的整数包括正整数、0 和负整数。用来存储人的年龄，出生年份，学生人数等整数相关的数据。

```python
number = 78
print(number)
print(type(number))
```

**数字分隔符：**为了提高数字的的可读性，Python3中允许使用下划线`_`作为数字的分隔符。通常每隔三个数字添加一个下划线，类似于英文数字中的逗号。下划线不会影响数字本身的值。

```python
click = 1_300_000
```



## 2. 浮点数类型

用来存储人的身高，体重，薪资等小数相关的数据，Python中的小数有两种书写形式。

```python
pi = 3.14
print(pi)
print(type(pi))
```

- 十进制形式：就是我们平时看到的小数形式，例如 34.6、346.0、0.346。

- 指数形式：2.1E5 = 2.1×10^5^
  
  > 只要写成指数形式就是小数，例如 14E3 等价于 14000.0



## 3. 字符串类型

用来存储人的名字，家庭住址，性别等描述性质的数据。字符串由单引号、双引号、多引号包裹。字符串的内容可以包含字母、标点、特殊符号、中文、日文等全世界的所有文字。

用单引号、双引号、多引号，都可以定义字符串，本质上是没有区别的，但是当字符串内容中出现引号时，我们需要进行特殊处理，否则Python会解析出错。

```python
str1 = 'I'm a great coder!'
```

上面字符串中包含了单引号，此时 Python 会将字符串中的单引号与第一个单引号配对，这样就会把`'I'`当成字符串，而后面的`m a great coder!'`就变成了多余的内容，从而导致语法错误。

对于这种情况，我们有两种处理方案：

- 在引号前面添加反斜杠`\`就可以对引号进行转义，让 Python 把它作为普通文本对待    

  ```python
  str1 = 'I\'m a great coder!'
  ```

- 使用不同的引号包围字符串

  ```python
  str1 = "I'm a great coder!"
  ```

转义字符：把字符串当中字符原本的意思给转变为新的意思。转义字符有时候会带来一些麻烦，例如我要表示一个Windows路径文件路径。因为`\`的特殊性，我们需要对字符串中的每个`\`都进行转义这种写法需要特别谨慎，稍有疏忽就会出错。

原始字符串：在字符串的开头加上`r`前缀，就变成了原始字符串`\`不会被当作转义字符，所有的内容都保持“原汁原味”的样子。
