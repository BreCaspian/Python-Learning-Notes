     ```python
     [170, 160, 180, 165]
     ```

     ```
     假设第一个就是最大值
     使用假设的和第二个进行比较, 发现更大的就替换假设的
     使用假设的和第三个进行比较, 发现更大的就替换假设的
     使用假设的和第四个进行比较, 发现更大的就替换假设的
     最后，假设的就是最大的.
     ```
